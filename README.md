# data-analyst-power-bi

Ecommerce Data Analysis & Dashboard

This project demonstrates a comprehensive end-to-end data processing and visualization workflow. It involves cleaning a large ecommerce dataset and building an interactive dashboard to track key business metrics.



🛠️ Data Cleaning & Preprocessing
Before visualization, the raw data underwent a rigorous cleaning process to ensure accuracy and consistency:


Duplicate Removal: Initial steps were taken to identify and remove all duplicate entries from the dataset to prevent skewed results.


Handling Null Values: Blank values in the dataset were replaced with the label "Unknown" to maintain data integrity during analysis.


Power Query Transformation: Advanced data shaping and transformations were performed using Power Query to prepare the data for the Power BI environment


📊 Dashboard Key Insights

The final Ecommerce Dashboard provides a high-level view of sales, inventory, and logistics performance:Total Revenue: The project identified a very high total revenue, calculated at $100.76$M.Inventory Management: The stock value is significant, totaling 25M units/value.Logistics: Shipping costs have been identified as a major financial contributor to the business operations.Tax and Discounts: Both Total Tax and Total Discount metrics are sitting at approximately 1M each.



📈 Visualizations Included

The interactive Power BI dashboard features several views to drill down into the data:Geographic Analysis: A treemap showing the sum of shipping costs across global locations like Dubai, Mumbai, and New York.Customer Demographics: A breakdown of popularity and discounts by gender (Male, Female, Non-Binary).Category Performance: Analysis of discounts and gender distribution across product categories such as Books, Apparel, and Footwear.Financial Health: A gauge chart visualizing the relationship between Price, Return Rate, and Shipping Costs.
